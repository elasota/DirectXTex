Convection Texture Tools are an enhanced fork of Microsoft's DirectX Texture Library (DirectXTex).
For usage, see ReadMe_DirectXTex.txt

Current features:
- "High quality mode" BC7 GPU compressor.  To enable, add -bchq to TexConv's command line, or add TEX_COMPRESS_HQ to the compression flags when using Compress.
